## TypeScript Deep Dive 日本語版について の写経です。
## https://typescript-jp.gitbook.io/deep-dive/


## 環境構築
・npm install -g typescript ts-node

## プロジェクト作成準備
* mkdir 00_hello
* cd ./00_hello

## プロジェクトの初期化
* npm init -y
* npm install typescript --save-dev
* npm install @types/node --save-dev
* npx tsc --init


## ビルド
* tsc

## 実行
* node main.js
